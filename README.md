# Copywriting Skills for AI Agents & Marketers

5 battle-tested persuasion patterns with fill-in-the-blank templates — designed for blog posts, LinkedIn content, email sequences, landing pages, and ad copy.

**Stop guessing. Start with a framework.**

## What's Inside

Each skill includes: psychology explanation → step-by-step framework → ready-to-use template → real-world examples → anti-patterns to avoid → combination playbooks.

| # | Skill | What It Does | Best For |
|---|-------|-------------|----------|
| 1 | [Binary Identity Framing](skills/binary-identity-framing.md) | Forces reader to choose a side — no gray area | CTAs, blog intros, social hooks |
| 2 | [Parallel Narrative Structure](skills/parallel-narrative.md) | Two stories: one thrives, one stagnates — reader picks | Case studies, email sequences, landing pages |
| 3 | [Future Pacing](skills/future-pacing.md) | Paints a vivid picture of the reader's transformed state | Sales pages, service pitches, product launches |
| 4 | [Scarcity Stacking](skills/scarcity-stacking.md) | Layers 2-3 types of urgency that compound | Limited offers, course launches, event promos |
| 5 | [Exclusivity Positioning](skills/exclusivity-positioning.md) | Makes reader feel selected, not sold to | Premium services, membership sites, B2B |

## Quick Start

## Example: Stacking Binary Identity + Future Pacing + Scarcity

> *"There are two types of marketers reading this: those who'll build a content system before Q3, and those who'll still be manually writing in December. If you want to be in the first group, registration closes Friday. 12 spots left."*

## How to Use With AI

Each skill file is structured for AI consumption:

```
## Why It Works    → Psychology (gives the AI context)
## The Framework   → Step-by-step (gives the AI structure)
## Template        → Fill-in-the-blank (gives the AI a starting point)
## Examples        → Real-world (gives the AI style reference)
## Anti-Patterns   → What not to do (gives the AI guardrails)
## Combine With    → Stacking playbook (gives the AI advanced moves)
```

Point your AI agent at any skill file and say *"write me a [blog post / LinkedIn post / email] using this pattern for [your topic].*"
