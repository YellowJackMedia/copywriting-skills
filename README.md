# Copywriting Skills for AI Agents & Marketers

5 battle-tested persuasion patterns extracted from real-world marketing — designed for use in blog posts, LinkedIn content, email sequences, landing pages, and ad copy.

Each skill is a standalone framework you can drop into any writing project.

## Skills

| Skill | What It Does | Best For |
|-------|-------------|----------|
| [Binary Identity Framing](skills/binary-identity-framing.md) | Forces reader to choose a side | CTAs, blog intros, social hooks |
| [Parallel Narrative Structure](skills/parallel-narrative.md) | Shows transformation vs. stagnation through two characters | Case studies, email sequences, landing pages |
| [Future Pacing](skills/future-pacing.md) | Paints a vivid picture of the reader's transformed state | Sales pages, service pitches, product launches |
| [Scarcity Stacking](skills/scarcity-stacking.md) | Layers multiple scarcity types for urgency | Limited offers, course launches, event promos |
| [Exclusivity Positioning](skills/exclusivity-positioning.md) | Makes reader feel selected, not sold to | Premium services, membership sites, B2B |

## How to Use

1. Pick the skill that matches your goal
2. Read the framework
3. Fill in the template with your specifics
4. Combine multiple skills for maximum effect (e.g., Binary Identity + Future Pacing + Scarcity)
